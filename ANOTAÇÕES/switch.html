Switch Case

É empregada quando desejamos atingir mais eficiência e melhor performance, substituindo, até mesmo, a estrutura condicional if...else, sempre que possível e sempre que a condição a ser avaliada for a mesma. Executa até encontrar um valor verdadeiro.

Objetivo: Utilizando o recurso de seleção múltipla - Switch...case, pesquisar o nome do mês de referência através do respectivo número do mês informado.

Métodos utilizados:

>Entrada:document.form(entrada de dados via formulário);

>Saída: document.form (saída de dados via formulário).

Importante

-Os campos do formulário serão identificados com o prefixo f_ => ex.: f_mes

-Os campos do JavaScript serão identificados com o prefixo js_ => ex.: js_mes
